PSD converted to an editable HTML/CSS certificate template.

Files:
- index.html — main certificate markup
- styles.css — responsive styling and print rules
- assets/bg.jpg — extracted watercolor background from PSD
- assets/signature.png — extracted signature
- assets/stamp.png — extracted stamp

Open index.html in a browser. Text can be edited directly in HTML.
For print/export, use the browser Print dialog and choose landscape / save as PDF.
